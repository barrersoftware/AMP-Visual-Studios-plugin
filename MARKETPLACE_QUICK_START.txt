• Check VSIX file size (13 KB is fine, max 50 MB)
• Verify you're signed in with correct Microsoft account
• Try again - sometimes it's just a timeout

Publisher account issues?
• Use your personal Microsoft account
• If using organization, ensure you have admin rights
• Contact marketplace support if blocked
